---

__Download and install dependencies__:

```
npm install
```

__Run it__:

```
npm start
```

or directly using node executable (port is optional, default 8000)

```
node index.js <port>
```
__Docker__:

```bash
docker pull totalplatform/totaldb
docker run -p 8000:8000 totalplatform/totaldb
````
